# Roadmap

## Milestone 1: Construction Cost Center Dashboard

- Dockerized local development
- PostgreSQL schema seed for construction ERP records
- Project cost center dashboard
- Project budget versus actual cost
- Material inventory status
- Purchase order overview
- Progress billing claim overview

## Milestone 2: Core ERP Transactions

- Project master data CRUD
- Work item budget CRUD
- Supplier and subcontractor master data CRUD
- Purchase order creation and approval status
- Material receiving and job-site usage
- Cost entry posting by project and work item

## Milestone 3: Construction ERP Modules

- Subcontractor contracts
- Subcontractor progress payments
- Owner progress billing
- Change orders
- Retention tracking
- Equipment usage and rental cost allocation
- Daily reports linked to project costs

## Milestone 4: Civil Engineering Domain Features

- Tunnel chainage progress records
- Grouting hole records
- Ground improvement zone records
- Quality inspection records
- Safety issue tracking

## Milestone 5: Production Readiness

- Authentication
- Role-based permissions
- Database migrations
- Test suite
- CI checks
